This is the version selector

Due to requirements of task, I decided to implement two versions of the list selector. In order to run this application, you need to run the mock server I created for this task. for that matter, use the following command (click on the command to copy): {{dockerCommand}} Or you can run the server manually by following the instructions in the README file.

List Versions

  • Version 1
    • loads resources as indicated in the task
    • does not implement pagination
    • works well when payloads are small
    • does not depend on any change from the server, since it processes the data "as is"
    • will cause performance issues once the payload grows
  • Version 2
    • loads resources using pagination
    • works well when payloads are small
    • does not depend on any change from the server, since it processes the data "as is"
    • will cause performance issues once the payload grows
